Usuarios en el Sistema

Aquí podrías mostrar una lista de usuarios registrados. Esta funcionalidad requeriría una ruta adicional en el backend.

Roles del Sistema
Administrador
  • Crear y gestionar pedidos
  • Gestionar materiales
  • Ver todos los pedidos
  • Asignar pedidos a empleados
  • Registrar nuevos usuarios
Empleado
  • Ver pedidos asignados
  • Actualizar estado de pedidos
  • Usar el optimizador
  • Ver materiales disponibles
Requisitos
  • Usuario: Único, sin espacios
  • Email: Válido y único
  • Contraseña: Mínimo 6 caracteres
  • Rol: Admin o Empleado
Consejos
  • Usa nombres de usuario cortos y fáciles de recordar
  • Las contraseñas se almacenan encriptadas
  • El usuario puede cambiar su contraseña después
  • Asigna el rol "Empleado" por defecto
